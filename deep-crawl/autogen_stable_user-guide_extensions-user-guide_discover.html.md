## 社区项目列表#

名称 | 包 | 描述
---|---|---
[autogen-watsonx-client](https://github.com/tsinggggg/autogen-watsonx-client) | [PyPi](https://pypi.org/project/autogen-watsonx-client/) | 适用于 [IBM watsonx.ai](https://www.ibm.com/products/watsonx-ai) 的模型客户端
[autogen-openaiext-client](https://github.com/vballoli/autogen-openaiext-client) | [PyPi](https://pypi.org/project/autogen-openaiext-client/) | 通过 OpenAI API 适用于 Gemini 等其他 LLM 的模型客户端
[autogen-ext-mcp](https://github.com/richard-gyiko/autogen-ext-mcp) | [PyPi](https://pypi.org/project/autogen-ext-mcp/) | 适用于 Model Context Protocol 服务器工具的工具适配器
[autogen-ext-email](https://github.com/masquerlin/autogen-ext-email) | [PyPi](https://pypi.org/project/autogen-ext-email/) | 用于生成和发送电子邮件的电子邮件代理
[autogen-oaiapi](https://github.com/SongChiYoung/autogen-oaiapi) | [PyPi](https://pypi.org/project/autogen-oaiapi/) | 构建在 AutoGen 之上的 OpenAI 风格 API 服务器
[autogen-contextplus](https://github.com/SongChiYoung/autogen-contextplus) | [PyPi](https://pypi.org/project/autogen-contextplus/) | 增强的 model_context 实现，具有自动摘要和截断模型上下文等功能。
[autogen-ext-yepcode](https://github.com/yepcode/autogen-ext-yepcode) | [PyPi](https://pypi.org/project/autogen-ext-yepcode/) | 使代理能够使用 [YepCode](https://yepcode.io) 的无服务器运行时在隔离的远程沙箱中安全地执行代码。
