# Core#

AutoGen core 提供了一种简便的方法来快速构建事件驱动的、可扩展的、弹性的分布式 AI 代理系统。代理是使用 [Actor model](https://en.wikipedia.org/wiki/Actor_model) 开发的。您可以在本地构建和运行代理系统，并在准备就绪时轻松迁移到云中的分布式系统。

AutoGen core 的主要功能包括：

异步消息传递

代理通过异步消息进行通信，支持事件驱动和请求/响应通信模型。

可扩展且分布式

通过跨组织边界的代理网络实现复杂场景。

多语言支持

目前支持 Python 和 Dotnet 互操作代理，更多语言即将推出。

模块化且可扩展

具有高度可定制性，支持自定义代理、内存即服务、工具注册表和模型库等功能。

可观察且可调试

轻松跟踪和调试您的代理系统。

事件驱动架构

构建事件驱动的、可扩展的、弹性的分布式 AI 代理系统。
