# autogen_agentchat#

该模块提供了 autogen_agentchat 包的主要入口点。它包括用于跟踪日志和事件日志的日志记录器名称，并检索包的版本。

TRACE_LOGGER_NAME _ = 'autogen_agentchat'_#
    

跟踪日志的日志记录器名称。

EVENT_LOGGER_NAME _ = 'autogen_agentchat.events'_#
    

事件日志的日志记录器名称。
